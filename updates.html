UPDATE HISTORY

全ゲームのバグ修正・バージョンアップ履歴を時系列でまとめています。

v1.2.0 UPDATE 2025.02.24 VOID RUNNER
NEW FEATURES
  • スタートボーナスを実装しました。
  • デイリーミッションを実装しました。
BUG FIXES & CHANGES
  • 弾が大きくなるとプレイヤーも大きくなる問題を修正。
  • 難易度により数値の表示が大きくなる問題を修正。
  • ホーム画面に選択中の難易度が表示されるようになりました。
  • レベルの上がり具合を調整しました。
  • ムーブの上がり具合を調整しました。
  • ゲームオーバー後の画面により細かい表示がされるようになりました。
  • 敵とボスのステータスを調整しました。
  • 難易度の調整を行いました。
v1.0.1 FIX 2025.02.17 VOID RUNNER
BUG FIXES
  • 一部環境でスキルが発動できない問題を修正。
  • スキルを一度しか発動できない問題を修正。
  • スキル発動後の見た目の問題を修正。
CHANGES
  • スキル発動可能時は緑色、ピース不足時は黄色に変わるよう仕様変更。
